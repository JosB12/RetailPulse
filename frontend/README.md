# Frontend

React 19 + TypeScript powered by Vite, TanStack Query, and shadcn/ui.

## Stack

- React 19 + TypeScript
- Vite
- TanStack Query — server state management
- shadcn/ui + Tailwind CSS — UI components and styling
- SignalR Client — real-time notifications

## Run Locally

```bash
npm install
npm run dev
```
